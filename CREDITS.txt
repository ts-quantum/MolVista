# Credits & Licensing

MolVista is a specialized tool for chemical trajectory analysis and visualization. This document lists the core technologies and third-party libraries that make this project possible.

## Core Frameworks
- **[PySide6](https://qt.io)** (LGPL v3): The primary GUI framework.
- **[QtPy](https://github.com)** (MIT): Compatibility layer for Qt bindings.

## 3D Rendering & Visualization
- **[PyVista](https://pyvista.org)** (MIT): High-level interface to the Visualization Toolkit (VTK).
- **[VTK](https://vtk.org)** (BSD): The underlying 3D graphics engine.
- **[PyVistaQT](https://github.com)** (MIT): Integration of PyVista interactors into Qt widgets.

## Scientific Computing
- **[NumPy](https://numpy.org)** (BSD): Essential for numerical array operations.
- **[SciPy](https://scipy.org)** (BSD): Used for spatial distance calculations and the Hungarian Algorithm (`linear_sum_assignment`).
- **[Pandas](https://pydata.org)** (BSD): Efficient management of tabular data.
- **[Matplotlib](https://matplotlib.org)** (PSF): Responsible for 2D plots and data visualization.

## Utilities
- **[Pyperclip](https://github.com)** (BSD): Cross-platform clipboard support.
- **[psutil](https://github.com)** (BSD): System resource and process management.

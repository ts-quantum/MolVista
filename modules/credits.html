MolVista Credits

MolVista is a specialized tool for the visualization and manipulation of chemical trajectories, structural alignment, and advanced export preparation.

Core Technologies

PySide6 & QtPy: The graphical user interface is built on the Qt framework, providing a high-performance desktop experience.
License: LGPL v3
PyVista & VTK: Powering the 3D rendering engine for molecular structures and surface meshes.
License: MIT / BSD
PyVistaQT: Provides seamless integration of PyVista 3D interactors into the Qt application environment.
License: MIT

Scientific Computing & Data

NumPy & SciPy: The foundation for mathematical operations, vector calculus, and the Hungarian Algorithm (Linear Sum Assignment) used for atom mapping.
License: BSD
Pandas: Used for efficient management and analysis of tabular chemical data.
License: BSD
Matplotlib: Responsible for rendering high-quality 2D plots and analytical charts within the application.
License: PSF

Utility Libraries

Pyperclip: Enables cross-platform clipboard support for exporting data strings.
License: BSD-3-Clause

Source Code: Available at https://github.com/ts-quantum/MolVista

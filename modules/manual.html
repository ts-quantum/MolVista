MolVista Manual

Integrated Graphical Interface for Reaction Path Analysis and Trajectory Splicing.

1. Data Import & Setup

Load Files: Go to File -> Load trj to import your XYZ trajectory files (e.g., IRC segments). They will appear in the File List on the left.

Drag & Drop: Simply drag a file from the list and drop it into geo_view_0 (left plotter) or geo_view_1 (middle plotter). The Energy Profile and the Atom Index List for each segment will automatically load below the viewers.

2. Alignment Strategies

Option A Automatic Align

Use Action -> Align for a fully automated connection of two segments.

  • Auto-Flip: The program automatically determines the chronological order (End-to-Start matching) and reverses segments if necessary.
  • Result: The aligned structure and merged energy profile are displayed in the third plotter (right).

Option B Align Masked (Manual Control)

Use Action -> Align Masked for complex reactions involving fragment changes.

  • No Auto-Flip: You must manually check the orientation. Use Right-click -> Reverse if needed.
  • Define Anchor: Highlight the "moving" atoms (fragments) in the atom list. The *unselected* atoms form the rigid anchor for the overlay.
  • Splicing: Enable "Splicing" and set the number of transition images (default: 30) to create smooth Phase-In/Phase-Out animations.

3. Interaction & Visualization

  • Navigation: Click any point on the Energy Profile to jump directly to that specific molecular geometry.
  • Animation: Use the Play/Pause buttons above the plotters to visualize the reaction path.
  • Action Log: All operations (RMSDs, rotations, energy offsets) are recorded in the bottom-left log widget for transparency.

4. Export & Context Menus

Geo-Plotter (Right-Click)

  • Export Formats: Save your results as .xyz, .pov (POV-Ray), Blender, .mp4, or .png.
  • Tools: Access helper scripts (e.g., Split.py), reverse trajectories, or change atom color schemes.

Quantum Chemistry Bridge (Split Script)

After exporting a merged trajectory, MolVista offers to generate a _split.py Python template.

  • Purpose: This script serves as a bridge to electronic structure theory (e.g., ORCA).
  • Workflow: It automatically breaks the trajectory into single frames and generates the necessary input files and shell scripts for batch processing.
  • Customization: Open the _split.py in any text editor to adjust the ORCA_HEADER (method, basis set) or charges before execution.

Energy Profile (Right-Click)

  • Export: Save the plot as a high-resolution .png or the raw data as a .txt file.
  • Copy: Copy energy values directly to the clipboard for external use in Excel or Origin.
Developer Tip: Use the Split.py script to convert your merged XYZ trajectory into a series of MOLDEN files for subsequent electronic structure analysis (e.g., ESP or Orbital localization).
